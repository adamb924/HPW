Export XML data

Note that data will be from the current project and approach.

You can choose what to export. Exporting strings provides a more readable output—for instance by providing a list of forms and language names, instead of just indices that point you to those forms and language names. Here is a guide:

  • Strings only. Easier to read; harder to repurpose the data. If you only export strings, not all of the interrelations between the levels of analysis are exported.
  • Indices only. Easier for a computer to read; harder for a human to read. This is the most efficient option for carrying out subsequent processing.
  • Strings and indices. Easy for a computer and a human to read. Less efficient for subsequent processing.







After clicking on the “Export” button you will be prompted for a filename.
